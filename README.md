# dende-softhouse-modelo-preditivo-world-cup-2026-equipe-lauro

## Como rodar

1. Instale as dependencias:

```bash
pip install -r requirements.txt
```

2. Abra e execute o notebook principal:

```bash
jupyter notebook notebooks/modelo_copa.ipynb
```

## Entregas da Task 3

Ao executar o notebook, a Task 3 gera:

- `outputs/ranking_probabilidades.csv`: ranking oficial agregado por selecao.
- `outputs/feature_importance.csv`: lista das variaveis mais importantes para o modelo.
- `outputs/previsoes_detalhadas_test.csv`: previsoes linha a linha do `test.csv`.
